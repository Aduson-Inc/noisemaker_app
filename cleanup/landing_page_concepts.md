# Landing Page Design Concepts - NOiSEMaKER

**Date:** January 6, 2026
**Goal:** Replace the "Gen Z Bold" design with something more modern, professional, and trustworthy.

**Constraints:**
- NO PURPLE (overused by AI)
- Must work on mobile AND desktop
- Professional music industry feel
- High-converting layout

---

## CONCEPT 1: "Midnight Studio"
**Vibe:** Premium recording studio aesthetic - dark, sophisticated, trustworthy

### Color Palette
| Role | Color | Hex |
|------|-------|-----|
| Background | Rich Black | #0A0A0B |
| Surface | Dark Gray | #141416 |
| Primary Accent | Emerald | #10B981 |
| Secondary | Warm White | #F5F5F4 |
| Muted Text | Cool Gray | #6B7280 |

### Layout Structure
```
┌─────────────────────────────────────────────────┐
│  [DooWopp Logo 4x]              [Sign In]       │  ← Larger logo, minimal header
├─────────────────────────────────────────────────┤
│                                                 │
│     [HUGE NOiSEMaKER LOGO - Main Focal Point]   │
│                                                 │
│         "Your music deserves to be heard."      │  ← Single powerful tagline
│                                                 │
│              [ Get Started → ]                  │  ← Emerald CTA button
│                                                 │
│         8 Platforms • AI Content • Analytics    │  ← Subtle feature pills
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│   ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐           │
│   │ IG  │  │ TW  │  │ TT  │  │ YT  │  ...      │  ← Platform icons in a row
│   └─────┘  └─────┘  └─────┘  └─────┘           │
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│  "Stop wasting hours on promo that doesn't      │
│   work. We post to 8 platforms. You make music."│
│                                                 │
│  ┌──────────────┐  ┌──────────────┐            │
│  │ 42-Day Cycle │  │ Zero Effort  │            │  ← Feature cards (dark surface)
│  └──────────────┘  └──────────────┘            │
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│         [ Choose Your Plan → ]                  │  ← Second CTA
│                                                 │
└─────────────────────────────────────────────────┘
```

### Typography
- **Headline:** Inter Black or Bebas Neue (large, impactful)
- **Body:** Inter Regular (clean, readable)
- **Accent:** Monospace for stats/numbers

### Key Elements
- Subtle gradient on emerald CTA (slight glow effect)
- Platform icons have a soft white glow on hover
- Smooth scroll animations
- Minimal text, maximum impact
- Logo is THE focal point

### Why It Works
- Dark themes convert well in 2025 (A/B tested)
- Emerald = growth, success, money (music industry associations)
- Recording studio aesthetic = professional credibility
- Clean layout = trust

### Typography
- **Headline:** Playfair Display or Libre Baskerville (elegant, premium)
- **Body:** Source Sans Pro (professional, readable)
- **Numbers:** Bold weight for impact

### Key Elements
- Warm gradient border on hero card (gold → orange → gold)
- Soft ambient glow behind logo
- Testimonial adds social proof early
- Stats use gold text on dark cards
- Feeling of warmth and trust

### Why It Works
- Gold = premium, success, achievement
- Warm tones = inviting, approachable
- Testimonial early = social proof
- Card-based layout = modern, organized

## CONCEPT 3: "Electric Blue"
**Vibe:** Modern tech startup - clean, bold, confident

### Color Palette
| Role | Color | Hex |
|------|-------|-----|
| Background | Near Black | #030712 |
| Surface | Dark Blue Gray | #111827 |
| Primary Accent | Electric Cyan | #06B6D4 |
| Secondary Accent | Coral/Salmon | #F97316 |
| Text | Pure White | #FFFFFF |

### Layout Structure
```
┌─────────────────────────────────────────────────┐
│                                                 │
│  [DooWopp Logo]                      [Sign In]  │
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│  NOiSEMaKER                                     │  ← Large logo, left-aligned
│  ═══════════                                    │    with cyan underline
│                                                 │
│  Automated social media                         │
│  for independent artists.                       │  ← Clean, simple copy
│                                                 │
│  [ Get Started ]  [ See How It Works ]          │  ← Dual CTA (cyan + outline)
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│  ┌─────────────────────────────────────────┐   │
│  │  ░░░░░░░░ ANIMATED WAVEFORM ░░░░░░░░░   │   │  ← Subtle audio visualizer
│  └─────────────────────────────────────────┘   │    animation (cyan)
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│  HOW IT WORKS                                   │
│                                                 │
│  1. Link Spotify    2. Pick Plan    3. Relax   │  ← 3-step process
│     [icon]             [icon]          [icon]   │
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│  PLATFORMS                                      │
│  [IG] [TW] [TT] [YT] [FB] [RD] [DC] [TH]       │  ← All 8 platform icons
│                                                 │
├─────────────────────────────────────────────────┤
│                                                 │
│       [ Choose Your Plan → ]                    │  ← Coral CTA for contrast
│                                                 │
└─────────────────────────────────────────────────┘
```

### Typography
- **Headline:** Space Grotesk or Outfit (modern, geometric)
- **Body:** Inter (clean tech standard)
- **Labels:** All caps, letter-spaced

### Key Elements
- Animated audio waveform in cyan (subtle, music-related)
- Dual CTA strategy (primary + secondary action)
- 3-step "How It Works" builds confidence
- Platform icons show breadth of service
- Coral accent for important CTAs (attention-grabbing)

### Why It Works
- Cyan + Coral = high contrast, energetic but not childish
- Tech startup aesthetic = modern, trustworthy
- 3-step process = reduces perceived complexity
- Waveform animation = music identity without being cheesy

## COMPARISON SUMMARY

| Aspect | Midnight Studio | Golden Hour | Electric Blue |
|--------|-----------------|-------------|---------------|
| **Vibe** | Sophisticated | Premium/Warm | Modern Tech |
| **Primary Color** | Emerald #10B981 | Gold #F59E0B | Cyan #06B6D4 |
| **Best For** | Professional artists | Premium feel | Younger tech-savvy |
| **Social Proof** | Minimal | Testimonial early | Process-focused |
| **Animation** | Subtle glows | Gradient borders | Waveform |
| **Typography** | Bold sans-serif | Elegant serif | Geometric sans |
